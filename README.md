RUN2PLAY
=========================================

# Project members
* Kenny Jesús Flores Huamán

## Description
Project written in Kotlin where firebase was used together with the google maps API, so that the user can run in exchange for fictitious rewards.

## Requirements
* Android Studio 3.6 or higher
